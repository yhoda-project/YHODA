# YHODA (Yorkshire & Humber Office for Data analytics)

The YHODA project provides a comprenhensive data integration and analytics platform for the Yorkshire and Humber region. 
It includesaggregated data insights and regional indicators, supporting the Yorkshire Engagement Portal (YEP) to facilitate data-driven policy and decision-making. 

# Key Components
- Data transformation: Integration and cleaning.
- Data classification. 
- Visualization and analytics tools.
- Regional policy insights for stakeholders.
